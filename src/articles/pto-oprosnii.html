Теплообменники пластинчатые разборные предназначены для передачи тепла между средами (жидкость или пар), которые проходят по каналам теплообменника. Для подбора теплообменника необходимы данные по каждой из сред, учавствующих в теплообмене. Заполнив опросный лист на теплообменник, Вы получите наиболее точный расчёт теплообменного аппарата под вашу задачу.

Какие данные необходимы для заполнения опросного листа?

  • Температурные графики

    Значения температур на входе и на выходе для каждой из сред.

  • Потери давления

    Значение в атмосферах, барах, которое "теряется" теплообменником в соответствии с требованиями по проекту.

  • Расходы

    Значения расходов как по греющей, так и по нагреваемой сторонам. Основное - значение расхода по нагреваемой стороне.

  • Мощность

    Тепловая нагрузка на теплообменный аппарат.

Это 4 основных блока данных.

Пластинчатый теплообменнник можно рассчитать по органиченным данным, к примеру, оттолкнуться от расхода жидкости по нагреваемой среде и температурных графиков. В таком случае программа расчета посчитает мощность теплообменника. Если оттолкнуться от мощности, то программа выдаст количестство жидкости, которое можно раходовать в определенный период времени(сокращенно - расход). В опросном листе на теплообменник Вы можете указать те параметры, которые знаете. Наши специалисты подскажут каких данных не хватает, можно ли их вычислить по имеющимся характеристикам.
